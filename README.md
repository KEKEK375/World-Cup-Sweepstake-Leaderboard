# Sweepstake Leaderboard for the FIFA 2026 World Cup!
This is just a fun little project I made for a sweepstake I'm part of where we each ahve multiple teams so this helps to keep track of who si doing the best overall!

## To change it for your own sweepstake:
- clone the repository
- change the sweepstake.json to contain your people and their countries (make sure they match up with the names [here](https://www.fifa.com/en/tournaments/mens/worldcup/canadamexicousa2026/standings))
- host it as on github pages (or elsewhere)!
